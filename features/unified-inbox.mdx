---

Zero's unified inbox gives you a single place to read and respond to email from every connected account. Instead of switching between browser tabs or apps, all your mail arrives in one consistent interface that you control.

## Folders

Zero organizes email into standard folders available for each connected account:

| Folder | Description |
|--------|-------------|
| Inbox | Incoming mail |
| Sent | Sent messages |
| Drafts | Unsent drafts |
| Starred | Starred/flagged threads |
| Archive | Archived threads |
| Spam | Spam/junk mail |
| Trash | Deleted messages |

## Reading threads

Click any thread in the inbox to open it. Zero displays the full conversation in chronological order. The thread view shows sender information, timestamps, and rendered HTML content. Attachments appear inline at the bottom of each message.

## Replying and composing

Use the reply button at the bottom of any thread to respond. To compose a new message, click the **Compose** button in the sidebar. Zero supports reply, reply-all, and forward actions.

<Tip>Press **C** to open a new compose window from anywhere in the inbox.</Tip>

## Search

Use the search bar at the top of the inbox to find messages. Zero searches across subject lines, sender names, and body content. You can combine keywords with label filters to narrow results.

## Multi-account switching

If you have connected multiple email accounts, use the account switcher in the navigation to change the active connection. Each account maintains its own folder structure and labels.

## Thread sync

Zero syncs your email using Cloudflare Durable Objects and R2 for fast reads. Threads are cached locally so folder navigation is near-instant. Use **Settings → Connections** to manage your connected accounts or trigger a manual sync.
