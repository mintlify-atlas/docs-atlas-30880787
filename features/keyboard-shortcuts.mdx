Zero supports keyboard shortcuts throughout the interface so you can manage email without reaching for the mouse. Shortcuts are grouped by the context in which they work: globally available, within the mail list, inside an open thread, and inside the compose window.

## Global shortcuts

These shortcuts work from anywhere in Zero.

| Keys | Action |
|------|--------|
| `C` | Compose a new email |
| `G` then `I` | Go to inbox |
| `G` then `D` | Go to drafts |
| `G` then `T` | Go to sent mail |
| `Ctrl`+`K` / `Cmd`+`K` | Open command palette |

## Thread display shortcuts

These shortcuts are active when you have an email thread open.

| Keys | Action |
|------|--------|
| `R` | Reply to email |
| `A` | Reply all |
| `F` | Forward email |
| `Cmd`+`Backspace` / `Meta`+`Backspace` | Move thread to bin |

## Mail list shortcuts

These shortcuts work when focus is on the email list.

| Keys | Action |
|------|--------|
| `Escape` | Exit selection mode |

## Compose shortcuts

These shortcuts work inside the compose window.

| Keys | Action |
|------|--------|
| `Ctrl`+`Enter` / `Cmd`+`Enter` | Send email |

<Note>On macOS, `Ctrl` shortcuts use `Cmd` instead. The `Meta` key refers to the `Cmd` key on macOS and the `Windows` key on Windows.</Note>

## Command palette

Press `Ctrl`+`K` (or `Cmd`+`K` on macOS) from anywhere to open the command palette. The palette gives you quick access to navigation, actions, and settings without memorizing individual shortcuts.

## Customizing shortcuts

Shortcut customization is not yet available in the UI. The full list of active shortcuts is defined in the application source and can be extended by contributors. Several shortcuts are currently stubbed out for future activation, including:

- `/` — Search
- `E` — Archive email
- `!` — Mark as spam
- `#` — Delete email
- `U` (mail list) — Mark as unread
- `R` (mail list) — Mark as read
- `Ctrl`+`A` — Select all emails

Check the [GitHub repository](https://github.com/Mail-0/Zero) for the latest shortcut definitions or to propose new shortcuts.
